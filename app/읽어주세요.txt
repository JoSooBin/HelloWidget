# 위젯 widget
안드로이드 화면(액티비티)를 구성하는데 사용하는 요소

#뷰
View클래스를 상속해서 만든 위젯들

#뷰그룹
View 클래스를 상속해서 만든 위젯들
뷰들을 적절한 위치에 배치하거나 여러 뷰들을 하나로 모으는데 사아ㅛㅇ
배치관리자 : 적절한 위치에 배치
뷰 컨테이너 : 여러 뷰들을 하나로 묶음

#텍스트뷰
사용자가 수정할 수 없는 텍스트를 표시하는 위젯

#에디트텍스트
사용자에게 문자열이나 숫자를 입력받을 수 있는 위젯
input
layout_width
hint

#버튼
사용자가 손으로 클릭할 수 있는 위젯
버튼을 통해 사용자가 어떤 결정을 선택하거나 다음 화면으로 전환하는 등의 인트랙티브한 응답을 할 수 있음
backgorundhint : 버튼의 배경색
textAllCaps : 버튼 텍스트의 영어 대문자 표시 여부

#이미지뷰
이미지를 화면에 보여주기 위한 위젯
src : 이미지 경로 지정
scaleType : 이미지를 어떻게 보여줄지 저장

#체크박스
사용자가 여러 항목들 중 하나이상의 항목을 선택할 수 있게 해주는 위젯
checked : 항목의 선택여부를 표시

#라디오버튼
사용자가 여러 항목들 중 하나의 항목만을 선택할 수 있게 해주는 위젯
checked : 항목의 선택여부를 표시

#토글버튼
사용자가 선택을 하면 커짐이 표시되고 다시 선택하면 꺼짐을 표시하는 위젯
textOff : 토글버튼이 커졌을때 표시할 문자열 지정
textOn : 토글버튼이 커졌을때 표시할 문자열 작성

#색상명 사용하기
위젯에 색상을 지정하려면  보통 16진수 코드를 이용함
단, 색상이름으로 색상을 지정하려면
values - clolrs.xml에 색상명과 16진수코드를 선언하고 '@color/색상명' 형태로 사용하면 됨

색상은 AABBCCDD 형태로 정의하거나 AA - 투명도, BB - 빨강, CC - 녹색, DD - 파랑으로 정의함함
